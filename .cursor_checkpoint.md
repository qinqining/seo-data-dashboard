### 1.2 Value 美元换算

- 函数：`script/sync.py` → **`normalize_ahrefs_usd`**（约 L1172）。
- 调用：`get_page_ahrefs_metrics` 读 `top-pages.value` → `ahrefs_value`；写入 `build_page_controls` → `format_mingdao_decimal(..., 2)`。
- 规则：`amount >= 100` → **按美分 ÷100**；`< 100` → 不除（边界：50 美分可能写成 50 而非 0.50）。
- 例：API `380` → **3.80**（$3.8）；API `25400` → **254.00**（$254）。

- 重跑：`run_sync.bat --only-link-building`
