### 单站流水线

```
共用 Google 凭据（整次 sync 一次 OAuth + 刷新重试）
  → 每站最多 3 次整站重试（443/SSL 瞬时错误）
GSCClient(site) + AhrefsClient(anchor)
  ├─ sync_keywords      → 站点关键词库
  ├─ sync_pages         → 页面管理表（仅更新已有 URL 行）
  ├─ sync_backlinks     → 外链监控表
  └─ sync_dashboard     → SEO 自动数据看板（7 天；锚点日含 Ahrefs 分档+周环比）
```

### 锚点日与周环比（详见 `docs/20260602report.md` §2）

| 概念 | 规则 |
|------|------|
| **锚点日** | 运行日 − `DATA_DELAY_DAYS`（默认 **3**），≠ 运行当天 |
| 看板 7 行 | 锚点−6 … 锚点（例：5/24～5/30） |
| **周环比流量** | GSC：5/24–30 点击合计 vs 5/17–23 |
| **周环比 Top 分档** | Ahrefs：`date`=锚点 vs `date_compared`=锚点−7（如 5/30 vs 5/23） |
| Top 分档 / 四档周环比 | **仅锚点日一行**；国家 = `AHREFS_TARGET_COUNTRY`（默认 **us**） |

### CLI

```bat
run_sync.bat
run_sync.bat --skip-keywords --skip-backlinks --skip-dashboard   REM 仅页面
run_sync.bat --skip-pages --skip-backlinks --skip-dashboard      REM 仅看板
run_sync.bat --site cncpioneer --site fecision ...               REM 跳过某站用省略
```

## 5. 方案已定、未开发：关键词分级（`docs/keyword-grading-scheme.md` v2）

- **分类**：仅 Ahrefs **搜索意图** 六类；主意图优先级：**交易 > 商业 > 导航 > 品牌 > 本地 > 信息**
- **不算分/不写**：产品词、工艺词、应用词、材质词、询盘词（「关键词类型」列废弃自动写入）
- **公式**：`价值分 = ((Vol/1000×60)+(CPC×80))×意图系数/(KD+20)×100` → **P0～P3 / 未分级**
- **待接明道云**：列「搜索意图」、可选「价值分」；优先级扩展 **P3**；`mingdao_worksheets.json` + sync 写表逻辑
