### 1.2 页面表 — 脚本能力与 Ahrefs 对照（已与 dashboard §4.3 同步）

| 能力 | 说明 |
| --- | --- |
| **import** | `addRow`：独立站 + 页面URL + 数据日期（GSC 展示≥阈值 + 首页 seed） |

| **Ahrefs UI** | Top keyword→主关键词；Volume→主词搜索量；Keywords→排名关键词数；Words→暂不写 |
