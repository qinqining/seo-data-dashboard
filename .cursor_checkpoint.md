| 密钥      | `.env`                                                                            |
| 缓存      | `cache/gsc/{site}.json`（仅 GSC；Ahrefs 当次运行内存缓存按日）                                  |
| 报告      | `reports/sync-report-*.txt`（含 **Summary 总结**）                                     |


### 单站流水线

```
共用 Google 凭据（整次 sync 一次 OAuth + 刷新重试）
  → 每站最多 3 次整站重试（443/SSL 瞬时错误）
GSCClient(site) + AhrefsClient(anchor)
  ├─ sync_keywords      → 站点关键词库（含 Intent/CPC/价值分，若 .env 已配）
  ├─ sync_pages         → 页面管理表
  ├─ sync_backlinks     → 外链监控表
  └─ sync_dashboard     → 看板 7 天 GSC + 每天 Ahrefs 四档；锚点日含 RD/收录/预警/周环比流量/周均值
```

### 锚点日与看板字段写入节奏


| 概念            | 规则                                                          |
| ------------- | ----------------------------------------------------------- |
| **锚点日**       | `today − DATA_DELAY_DAYS`（默认 3）或 `--anchor-date YYYY-MM-DD` |
| 看板 7 行        | 锚点−6 … 锚点（例 5/24～5/30）                                      |
| **按日 GSC**    | 自然点击、展示量、平均CTR、全站加权平均排名                                     |
| **按日 Ahrefs** | Top1-3～Top21-100、周环比Top1-3～Top21-100（每行 date / date−7）      |
| **仅锚点日**      | Backlinks变化、收录/异常、异常预警、周环比流量、周平均排名、周自然点击                    |
| Ahrefs 国家     | `AHREFS_TARGET_COUNTRY`（默认 **us**）                          |


### 常用 CLI

```bat
REM 仅看板，锚点 5/30，清缓存后建议加 --refresh
run_sync.bat --skip-keywords --skip-pages --skip-backlinks --anchor-date 2026-05-30 --refresh

REM 补失败站
run_sync.bat --skip-keywords --skip-pages --skip-backlinks --anchor-date 2026-05-30 --site fecision

REM 删 GSC 本地缓存
del /q D:\seo-data-dashboard\cache\gsc\*.json
```
