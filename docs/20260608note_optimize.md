## 1. 先把站做好，看哪几张表？

### 价值排序（由高到低）


| 梯队         | 表                | 一句话价值                             |
| ---------- | ---------------- | --------------------------------- |
| **★★★ 必看** | **GSC Top 查询明细** | 用户 **真实** 搜什么、有没有点击（改内容/选词的第一依据）  |
| **★★★ 必看** | **页面管理表**        | **执行台账**：哪些 URL 有流量、收没收录、优化有没有记录  |
| **★★★ 必看** | **站点关键词库**       | Ahrefs **排名资产**：P0/P1、排名变化、该词落在哪页 |
| **★★ 补充**  | **GSC Top 页面明细** | 自动发现 **哪页在哪个国家有流量**（找新页、找国家机会）    |
| **★★ 补充**  | **SEO 自动数据看板**   | 全站 **健康度**：涨跌、异常预警、DR、四档词数        |
| **★ 中长期**  | **外链建设表**        | 审外链、RD；偏权重建设，不解决「这周改哪篇文章」         |
| **流程**     | 每日执行日志 / 周绩效复盘   | 不产数据，保证 **动作落地、可复盘**              |


### 谁回答什么问题

```
用户搜什么、点没点？       → GSC Top 查询
流量落在哪些页？           → GSC Top 页面
我重点管哪些页、改没改？   → 页面管理表
词库排名怎样、先做哪个？   → 关键词库
全站好不好、有没有异常？   → 看板
外链够不够、合不合格？     → 外链表
```

**若时间紧，只盯两张：GSC Top 查询 + 页面管理表。**

## 2. 每周怎么用（建议节奏）


| 时间         | 做什么      | 看哪张表                   |
| ---------- | -------- | ---------------------- |
| **周一**     | 扫全站有没有异常 | 看板 → 异常预警 ≠ 正常         |
| **周一**     | 定本周主攻词   | GSC Top 查询（usa + 点击降序） |
| **日常**     | 改页、记优化   | 页面管理表                  |
| **每周**     | 对词库排优先级  | 关键词库 → P0/P1 且排名 >30   |
| **有外链同事时** | 审 RD     | 外链表                    |


**标准闭环：** GSC Top 查询找词 → 关键词库定优先级 → 页面管理表改 URL 并填「优化记录」。

## 3. 各表怎么查才有用信息

### 3.1 GSC Top 查询明细（最优先）

**表是什么：** 每个 **词 × 国家 × 日期** 一行；数据来自 Google **真实** 搜索。

**常用筛选（明道云视图）：**


| 想看什么        | 筛选 / 排序                                        |
| ----------- | ---------------------------------------------- |
| 美国主战场 Top 词 | `独立站` = 本站 · `国家` = **usa** · 按 **点击** 降序      |
| 有曝光没点击（可优化） | `usa` · **展示量 ≥ 20** · **点击 = 0** · 看 **平均排名** |
| 差一点进首页      | `usa` · **平均排名 11～30** · 点击 > 0 或展示较高          |
| 已在首页但 CTR 低 | `usa` · **平均排名 ≤ 10** · **展示 ≥ 50** · CTR 偏低   |


**读数注意：**

- 表里是 **单日** 快照；**1～2 次展示** 的行波动大，**别单独下结论**，多看几天或汇总 7 日。
- **GSC 为准**；和 Ahrefs 关键词库排名不一致是正常的。

**两行示例怎么读：**


| 情况  | 展示  | 点击  | 排名  | 含义            | 要不要做             |
| --- | --- | --- | --- | ------------- | ---------------- |
| A   | 20  | 0   | 91  | 有曝光但排很后，几乎没人点 | 词有价值则值得优化；看趋势    |
| B   | 1～2 | 0   | 1   | 极少几次碰巧排前，样本太小 | **先观察**，等展示量上来再评 |


---

### 3.2 页面管理表（执行中心）

**表是什么：** 重点 URL 台账 = GSC 真实表现 + Ahrefs 参考 + 运营手工字段。

**常用筛选：**


| 想看什么   | 筛选 / 排序                           |
| ------ | --------------------------------- |
| 先修收录   | **收录状态** = 未收录 / 索引异常             |
| 高展示低转化 | **GSC展示量** 降序 · 点击很少              |
| 本周要改的页 | **运营页面类型** 已填 · **优化日期** 为空或很久没更新 |
| 已优化待观察 | **优化记录** 有内容 · 对比 GSC 点击/排名变化     |


**运营必填：** 运营页面类型、优化日期、优化记录（脚本不会写）。

**和 GSC Top 页面的区别：** 本表 = **你管的页**；GSC Top 页面 = **全站自动 Top 页发现**，互补不替代。

---

### 3.3 站点关键词库（词策略）

**表是什么：** Ahrefs 认为你站 **有排名的词**（默认 Top 1000）；运营填优先级和落地页。

**常用筛选（文档 §4.2 视图建议）：**


| 视图名           | 条件                                      |
| ------------- | --------------------------------------- |
| **P0/P1 待优化** | 优先级 = P0 或 P1 · **当前排名 > 30**           |
| **已进 Top10**  | 当前排名 ≤ 10                               |
| **词页不一致**     | **排名落地页** ≠ **关联落地页**（计划改的页和实际排名页不是同一个） |


**怎么用：** 搜索量、KD、CPC 看「值不值得做」；**当前排名 + 排名变化** 看进度；和 GSC Top 查询对照——GSC 有流量但词库没有 = 长尾机会。

---

### 3.4 GSC Top 页面明细（发现流量页）

**表是什么：** 每个 **URL × 国家 × 日期** 一行；GSC 自动 Top 页。

**常用筛选：**


| 想看什么     | 筛选 / 排序                         |
| -------- | ------------------------------- |
| 美国流量页    | `独立站` + **usa** · 按 **点击** 降序   |
| 某国有没有页在跑 | 选目标 **国家** · 点击 > 0             |
| 新发现的页    | 点击高但 **页面管理表** 里还没有 → 补进页面表并标类型 |


---

### 3.5 SEO 自动数据看板（健康检查）

**表是什么：** 每站每天一行；全站汇总，**不能**代替词级、页级分析。

**常用用法：**


| 看什么    | 怎么用                              |
| ------ | -------------------------------- |
| 有没有出事  | **异常预警** ≠ 正常 · 日期降序             |
| 流量趋势   | **自然点击**、**周环比流量**（锚点日行）         |
| 排名资产趋势 | Top1-3～Top21-100 词数 · 各档 **周环比** |
| 权重     | **本站DR** · **Backlinks变化**       |


**注意：** 「全站加权平均排名」≠ 某个词的排名；≠ GSC Top 查询里的「平均排名」。

---

### 3.6 外链监控表（中长期）

**常用筛选：** `审核结果` = 合格 · `外链存活` = 有效 · **域名DR ≥ 30**（合格 RD 视图）。

**用途：** 审外链质量、统计本周合格 RD；和内容改站不同频，有专人做外链时再看。

## 4. 三张核心表对照（避免混读）


| 维度   | GSC Top 查询       | 关键词库                | 页面管理表                  |
| ---- | ---------------- | ------------------- | ---------------------- |
| 数据性质 | Google **真实** 流量 | Ahrefs **估算/模型** 排名 | GSC 真 + Ahrefs 参考 + 人工 |
| 粒度   | 词 × 国家           | 一词一行                | 一 URL 一行               |
| 最适合  | 这周用户搜啥、点没点       | 词库优先级、排名进度          | 改哪页、记优化、看收录            |
| 决策权重 | **流量决策以 GSC 为准** | 补市场体量/KD            | **动作落在这里**             |


---

## 5. 三条实操口诀

1. **找词** → GSC Top 查询 · usa · 点击降序；再看「展示高、排名 11～30、点击少」
2. **定优先级** → 关键词库 · P0/P1 · 排名 >30
3. **动手改** → 页面管理表 · 填类型 + 优化记录 · 盯收录和 GSC 四指标

看板每周扫一遍异常；GSC Top 页面用来 **发现** 还没进页面表的高流量 URL。
