# 飞书 SEO 表格：API 对照与运营填写说明

本文档供 **运营经理** 与 **方案评审** 使用，说明：

1. GSC / Ahrefs **实际能同步哪些数据**
2. 表格字段与 API **是否一致、是否合理**
3. **建议修改** 的字段名或备注
4. 运营在同步前 **必须填写** 的内容

同步方式：每周双击 `run_sync.bat` 一次。脚本 **只更新已有行**，不会自动新建关键词/页面/外链记录。

---

## 一、GSC / Ahrefs 当前可调用的数据

| API | 已接入能力 | 写入哪些表 |
|-----|------------|------------|
| **GSC Search Analytics** | 全站点击、展示、CTR、平均排名；单 URL 点击；7 天点击汇总（周环比） | 看板、页面表 |
| **GSC URL Inspection** | 单 URL 收录状态（已收录 / 未收录 / 索引异常） | 页面表；看板汇总收录数 |
| **Ahrefs organic-keywords** | 站点 organic 词排名、搜索量、KD、7 日排名变化 | 关键词表；看板 Top10/Top30 |
| **Ahrefs keywords-explorer/overview** | 未进 organic 的词仍可查搜索量、KD | 关键词表 |
| **Ahrefs refdomains-history** | 近 7 天 referring domains 变化 | 看板 |
| **Ahrefs domain-rating** | 来源域名 DR | 外链表 |
| **Ahrefs all-backlinks** | 来源域名是否仍有 live 外链 | 外链表 |

**当前未接入（PPR 有提但脚本不写）**

| 能力 | 说明 |
|------|------|
| GSC 全站收录总数 | API 无法直接给出「全站已收录页面总数」 |
| GSC Coverage / 抓取错误报告 | 无法一键同步全站 404、抓取失败总数 |
| 关键词「优化次数」 | 需关联页面表统计，尚未实现 |
| SEO 每日执行 / 周绩效复盘 | 以人工填写为主 |

## 二、逐表字段对照（一致性 + 合理性）

图例：**✅ 一致**　**⚠️ 可用但名实不符/有前提**　**❌ 未实现**

### 2.1 SEO 自动数据看板

| 字段名 | 实际来源 | 一致性 | 合理性 | 说明 |
|--------|----------|--------|--------|------|
| 日期 | 脚本 | ✅ | ✅ | 表示该行数据所属日期（非点击同步时间） |
| 自然点击（GSC） | GSC Search Analytics | ✅ | ✅ | 指定日期的全站自然点击 |
| 展示量（GSC） | GSC Search Analytics | ✅ | ✅ | 指定日期的展示次数 |
| 平均 CTR | GSC Search Analytics | ✅ | ✅ | 飞书字段类型应为 **百分比**；GSC 返回 0~1 小数 |
| 全站平均排名 | GSC Search Analytics | ⚠️ | ⚠️ | API 为 **全站加权平均排名**，不是「核心关键词平均排名」 |
| 核心词 Top10 数量 | Ahrefs + 关键词表 | ⚠️ | ✅ | 只统计 **关键词表已填写的词**；词表空则恒为 0 |
| 核心词 Top30 数量 | Ahrefs + 关键词表 | ⚠️ | ✅ | 同上 |
| 今日新增 RD（referring domain） | Ahrefs refdomains-history | ⚠️ | ⚠️ | 实际为 **近 7 天 RD 净值变化**，不是单日新增 |
| 网站总收录页面 | GSC URL Inspection 汇总 | ⚠️ | ⚠️ | 实为 **页面表已填 URL 中「已收录」数量**，非 Google 全站收录总数 |
| 抓取异常数量 | GSC URL Inspection 汇总 | ⚠️ | ⚠️ | 实为 **已监控 URL 中未收录/异常数**，非 GSC 覆盖率报告全站异常 |
| 异常预警 | 脚本规则 | ⚠️ | ✅ | 初版规则（周环比流量、页面收录等），可后续细化 |
| 周环比流量 | GSC Search Analytics | ✅ | ✅ | 本周 7 天点击 vs 上周 7 天 |
| 周环比 Top30 词 | Ahrefs + 关键词表 | ⚠️ | ✅ | 基于词表 + 7 日前排名对比，非全站 organic 词 |

**补充**：GSC 单日数据常有延迟，指定日点击为 0、但周合计有数，属正常现象。

### 2.3 页面全生命周期管理表

| 字段名 | 实际来源 | 一致性 | 合理性 | 说明 |
|--------|----------|--------|--------|------|
| 页面 URL | 人工 | — | ✅ | **同步前必填**；须为完整 `https://...` |

## 三、建议更改清单（字段名 / 备注）

以下为 **建议** 调整，不改也能跑通同步；改备注或字段名可减少误解。

| 优先级 | 表 | 现字段名 | 建议 | 原因 |
|--------|-----|----------|------|------|
| **高** | 看板 | 全站平均排名 | 备注改为「全站加权平均排名（GSC）」 | 与 PPR「核心词平均排名」不符 |
| **高** | 看板 | 今日新增 RD | 改名为 **「近 7 天 RD 变化」** 或加备注 | API 为 7 日差值，非单日 |
| **高** | 看板 | 网站总收录页面 | 改名为 **「已监控 URL 收录数」** 或加备注 | 非 Google 全站收录总数 |
| **高** | 看板 | 抓取异常数量 | 加备注「= 已监控 URL 未收录/异常数」 | 非 GSC 覆盖率全站异常 |
| **中** | 外链表 | 外链收录状态 | 改名为 **「外链存活状态（Ahrefs）」** 或加备注 | 避免与 Google 收录混淆 |
| **中** | 关键词表 | 优化次数 | 标为 **人工/暂不自动** 或二期再做 | 脚本未实现 |
| **低** | 看板 | 核心词 Top10/Top30 | 加备注「仅统计关键词表已填词」 | 避免以为全站所有词 |
| **低** | 看板 | 周环比 Top30 词 | 加备注「基于关键词表 + Ahrefs 7 日对比」 | 口径说明 |

**字段类型核对（飞书侧）**

| 字段 | 建议类型 |
|------|----------|
| 日期 | 日期 |
| 平均 CTR、周环比流量、周环比 Top30 词 | 百分比 |
| 自然点击、展示量、排名、DR、页面流量等 | 数字 |
| 收录状态、外链收录状态、异常预警 | 单选（选项与脚本写入值一致） |

## 四、表格设计总体评价

**合理之处**

- 看板 + 关键词 + 页面 + 外链 + 日志/复盘，结构清晰，符合 SEO 运营闭环。
- 自动字段与人工字段分离，方向正确。
- 各表用「日期」标记 API 快照，适合每周手动同步。

**主要风险（非推翻方案，而是备注要说清）**

1. 看板部分字段 **名称偏「全站/单日」**，实际 API 是 **监控子集 / 7 日窗口**。
2. Top10/Top30、收录数、异常数 **依赖运营先填词表/页面表**，否则看板数字长期为 0。
3. 「优化次数」等 PPR 自动字段 **尚未落地**，需在表内标注避免考核误解。

## 五、运营填写说明（同步前必做）

### 5.1 总览

| 表 | 运营必须先填 | 脚本自动写入 |
|----|--------------|--------------|
| SEO 自动数据看板 | 不用填 | 全部自动 |
| 核心关键词总库 | **关键词内容** | 日期、搜索量、KD、当前排名、排名波动 |
| 页面全生命周期管理表 | **页面URL** | 日期、页面流量、收录状态 |
| 外链建设管控表 | **外链来源域名** | 日期、域名DR值、外链收录状态 |

空行会被跳过，不算失败；详见 `reports/sync-report-*.txt` 中 `[WRITE]` / `[SKIP]`。

---

### 5.2 关键词 — 需要提供什么

**必填：关键词内容**（一行一词，英文商业词）

示例：`CNC machining`、`custom turning parts`、`5 axis machining`

**建议**

- 优先录入 P0/P1 核心询盘词（看板 Top10/Top30 只统计这些词）。
- 不要多个词挤在一格。
- 类型、优先级、绑定 URL 等由运营维护，脚本不覆盖。

---

### 5.3 页面 — 需要提供什么

**必填：页面URL**（完整链接）

✅ `https://www.cncpioneer.com/`  
✅ `https://www.cncpioneer.com/products/cnc-milling`  
❌ 只填 `/products/xx` 或 `cncpioneer.com/xx`

**建议**：每行一个 URL；首页、重点产品页、询盘页优先录入。

---

### 5.4 外链 — 需要提供什么

**必填：外链来源域名**（域名或完整 URL）

✅ `example-industry.com`  
✅ `https://www.example-industry.com/article/123`

**建议**：发布一条外链填一行；DR≥30 可作为内部合格线（审核仍由主管在「审核结果」填写）。

## 六、常见问题

**Q：报告里「N 行为空行已跳过」是失败吗？**  
A：不是。只表示有空行未填必填项；**已填的行会正常更新**。

**Q：页面 URL 已填，为什么还有 SKIP？**  
A：通常是 **其他空行** 被跳过；请看报告 `[WRITE]` 是否已有该 URL。

**Q：收录状态「已收录」谁填的？**  
A：脚本按 Google 检测 **自动写入**；请勿手改「日期 / 页面流量 / 收录状态 / DR / 排名」等自动列。

**Q：看板 Top10/Top30 一直是 0？**  
A：请先在关键词表填写 **关键词内容**，并确认 Ahrefs 能查到该词排名。
